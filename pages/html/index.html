基础 ​

HTML是用来描述网页的一种语言，超文本标记语言（Hyper Text Markup Language）。HTML标签参考手册。W3School-HTML教程

一个简单的HTML实例解析，关于HTML的元素、标签、属性、块元素、内联元素（行内元素）等内容从上方链接自行学习或查找。

html
<!-- 这是HTML的注释 -->
<!doctype html>
<!-- 声明为HTML5文档 -->
<html>
  <!-- 完整的html页面开始 -->
  <head>
    <!-- 头部元素开始 -->
    <!-- 通常包含以下标签 -->
    <!-- meta标签：定义关于 HTML 文档的元数据。元数据是关于数据的数据（信息） -->
    <meta charset="utf-8" />
    <!-- 设置字符集 -->
    <meta name="description" content="网页概述" />
    <!-- 网页概述，和SEO有关 -->
    <meta name="keywords" content="关键词" />
    <!-- 网页关键词，和SEO有关 -->
    <meta name="author" content="作者" />
    <!-- 网页作者，和SEO有关 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <!-- 视口设置（在后续css-响应式-视口有说明） -->
    <title>HTML实例解析</title>
    <link rel="stylesheet" href="styles.css" />
    <!-- 外部样式表 -->
  </head>
  <!-- 头部元素结束 -->
  <body>
    <!-- HTML 元素 -->
    <!-- 标准的HTML元素必须有开始标签和结束标签（自闭合标签、空元素标签等除外）-->
    <!-- 使用<br>时建议使用用<br />这种形式，所有元素都保持闭合 -->
    <div>
      <!-- 开始标签 -->
      content
      <!-- 元素的内容 -->
    </div>
    <!-- 结束标签 -->
    <!-- HTML 元素结束 -->
    <div />
    <!-- 自闭合标签 -->
    <img src="https://example.png" />
    <!-- 图片标签 -->
  </body>
</html>
<!-- 完整的html页面结束 -->

Canvas & SVG ​

更多细节请「点击这里」

  • <canvas>元素一定要设置宽高。理解Canvas的坐标。
  • <canvas>元素本身不具备绘图功能，必须使用JavaScript进行绘制
    • canvas元素实例拥有的方法
      • getContext('2d')：获取绘图上下文对象，用于进行2D绘图操作。
      • fillRect(x, y, width, height)：绘制填充矩形。
      • strokeRect(x, y, width, height)：绘制矩形边框。
      • clearRect(x, y, width, height)：清除指定区域内的图像。
      • beginPath()：开始绘制路径。
      • moveTo(x, y)：将路径移动到指定的坐标点。
      • lineTo(x, y)：绘制一条直线到指定的坐标点。
      • arc(x, y, radius, startAngle, endAngle, anticlockwise)：绘制弧形或圆。
      • closePath()：闭合路径。
      • fill()：填充当前路径。
      • stroke()：绘制当前路径的边框。
      • drawImage(image, x, y)：绘制图像。
      • save()：保存当前绘图状态。
      • restore()：恢复上一个保存的绘图状态。
    • canvas元素实例拥有的属性:
      • width：画布宽度。
      • height：画布高度。
      • style：CSS 样式属性，用于设置画布的样式。
      • fillStyle：填充颜色或样式。
      • strokeStyle：边框颜色或样式。
      • lineWidth：边框宽度。
      • globalAlpha：全局透明度。
      • globalCompositeOperation：设置绘图的合成模式。
      • font：字体样式。
      • textAlign：文本水平对齐方式。
      • textBaseline：文本垂直对齐方式。
html
<h2>绘制矩形 Canvas</h2>

<!-- 创建一个 canvas 元素 -->
<canvas id="myCanvas" width="300" height="200"></canvas>

<h2>绘制矩形 SVG</h2>

<!-- 创建一个 SVG 元素 -->
<svg id="mySvg" width="300" height="200">
  <!-- 使用 rect 元素绘制矩形 -->
  <rect
    x="50"
    y="50"
    width="200"
    height="100"
    fill="red"
    stroke="blue"
    stroke-width="2"
  />
</svg>

<script>
  // 获取 canvas 元素
  const canvas = document.getElementById("myCanvas");

  // 检查浏览器是否支持 canvas
  if (canvas.getContext) {
    // 获取绘图上下文
    const ctx = canvas.getContext("2d");

    // 设置矩形的填充颜色
    ctx.fillStyle = "red";

    // 绘制一个矩形（左上角坐标为 (50, 50)，宽度为 200，高度为 100）
    ctx.fillRect(50, 50, 200, 100);

    // 设置矩形的边框颜色和宽度
    ctx.strokeStyle = "blue";
    ctx.lineWidth = 2;

    // 绘制一个矩形的边框
    ctx.strokeRect(50, 50, 200, 100);

    // 清除指定区域的矩形
    ctx.clearRect(60, 60, 180, 80);
  } else {
    // 如果浏览器不支持 canvas，则显示错误信息
    console.log("抱歉，您的浏览器不支持 HTML5 Canvas。");
  }
</script>

<script>
  // 获取 SVG 元素
  const svg = document.getElementById("mySvg");

  // 检查浏览器是否支持 SVG
  if (typeof SVGRect !== "undefined") {
    // 使用 SVGRect 对象进行矩形操作
    const rect = document.createElementNS("http://www.w3.org/2000/svg", "rect");
    rect.setAttribute("x", "50");
    rect.setAttribute("y", "50");
    rect.setAttribute("width", "200");
    rect.setAttribute("height", "100");
    rect.setAttribute("fill", "red");
    rect.setAttribute("stroke", "blue");
    rect.setAttribute("stroke-width", "2");

    // 将矩形添加到 SVG 中
    svg.appendChild(rect);
  } else {
    // 如果浏览器不支持 SVG，则显示错误信息
    console.log("抱歉，您的浏览器不支持 HTML5 SVG。");
  }
</script>

无障碍访问 ​

更多细节请「点击这里」

HTML的可访问性（Accessibility）是指在设计和开发网页时，确保所有用户，包括身体上或认知上有障碍的用户，都能够轻松地访问和使用网页的能力。通过遵循一些通用的可访问性原则和使用特定的HTML元素和属性，可以提高网页的可访问性。

以下是一些提高HTML可访问性的方法：

  1. 使用语义化的HTML：使用正确的HTML元素来标记内容，以便屏幕阅读器和其他辅助技术可以正确地解读和呈现页面内容。例如，使用<h1>到<h6>元素来标记标题，使用<p>元素来标记段落，使用<nav>元素来标记导航等。
  2. 提供有意义的文本描述：为所有非文本元素（如图像、链接、按钮等）提供有意义的文本描述，以便无法直接看到页面内容的用户可以理解其含义。使用alt属性为图像提供替代文本描述，使用title属性为链接和工具提示提供描述。
  3. 使用适当的表单标记：为表单元素使用正确的标记，以便辅助技术可以正确地解释和导航表单。例如，使用<label>元素来关联表单控件和其标签，使用<fieldset>和<legend>元素来组织和描述相关的表单元素。
  4. 提供可键盘导航：确保网页可以通过键盘进行完全导航和操作，而不依赖于鼠标。这对于使用屏幕阅读器或无法使用鼠标的用户非常重要。使用tabindex属性来定义可键盘导航的顺序，并确保通过键盘可以激活所有交互元素。
  5. 考虑颜色对比度：确保网页中的文本和背景颜色之间有足够的对比度，以便视力较差的用户可以轻松阅读内容。可以使用在线工具来检查颜色对比度，并使用适当的颜色组合。
  6. 提供清晰的导航和结构：确保网页有清晰的导航和结构，使用户能够快速定位和访问所需的内容。使用标题、列表、导航链接等来组织页面，并使用正确的标记来表示页面结构。
  7. 使用ARIA属性：ARIA（Accessible Rich Internet Applications）属性是一组用于增强可访问性的HTML属性。通过使用ARIA属性，可以为一些交互元素（如菜单、对话框、标签页等）提供更详细的描述和交互性。例如，使用role属性来定义元素的角色，使用aria-label属性来提供元素的描述。

这些只是提高HTML可访问性的一些常见方法，实际上，可访问性是一个广泛的主题，还涉及到许多其他方面。为了确保网页的可访问性，建议参考Web Content Accessibility Guidelines（WCAG），这是一组国际标准，旨在帮助开发人员创建可访问的网页。

SEO基础 ​

更多细节请「点击这里」

SEO（Search Engine Optimization，搜索引擎优化）是一系列用于提升网站在搜索引擎排名中的技术和策略。通过优化网站内容和结构，以及增加外部链接和社交媒体信号等方式，SEO旨在提高网站在搜索引擎结果页面（SERP）中的可见性和排名，从而增加有机（非付费）流量。

网页开发者可以从以下几个方面着手进行SEO优化：

  1. 关键词优化：在网页的标题、头部标签（如H1、H2）、正文和图片的Alt属性等位置合理地使用关键词。这有助于搜索引擎理解网页内容并提高相关性，提升网页在搜索结果中的排名。
  2. 内容质量和原创性：创建高质量、有价值的内容对于SEO非常重要。确保网页内容丰富、有深度，并满足用户的需求。避免复制其他网页的内容，尽量提供原创的、独特的资料。
  3. 网页结构和导航：确保网页的结构清晰，使用语义化的HTML标签组织内容，包括使用合适的标题标签、段落、列表等。提供清晰的导航菜单和内部链接，使用户和搜索引擎能够方便地浏览和索引网页。
  4. 网页加载速度：优化网页的加载速度对于SEO和用户体验都至关重要。确保网页的代码精简，压缩CSS和JavaScript文件，优化图像大小，使用缓存和CDN等技术来加快网页加载速度。
  5. 移动友好性：确保网页在移动设备上具有良好的用户体验。使用响应式设计或移动优化的布局，以适应不同屏幕尺寸。确保网页内容和功能在移动设备上可用，并优化移动页面的加载速度。
  6. URL优化：使用简洁、描述性的URL，包含关键词，并避免使用动态参数。静态、短小、易于理解的URL有助于搜索引擎和用户理解网页内容。
  7. 外部链接和内部链接：积极获取高质量的外部链接，这有助于提升网页的权威性和排名。同时，在网页中合理使用内部链接，将相关内容连接起来，有助于搜索引擎发现和索引网站的更多页面。
  8. 元数据优化：使用合适的元数据，包括页面的标题（title）、描述（meta description）、关键词（meta keywords）等。这些元数据对搜索引擎的理解和排名起到重要作用，同时也能吸引用户点击搜索结果。
  9. 社交媒体整合：将网页与社交媒体账号整合，通过分享按钮和社交媒体链接，促进内容的分享和传播，提升网页的曝光度和流量。
  10. 网页安全性：确保网页的安全性，使用HTTPS协议来加密数据传输。安全的网页对搜索引擎和用户都更可信，也有助于提高排名。

需要注意的是，SEO是一项长期的工作，你需要持续地监测和优化你的网站，以适应搜索引擎算法的变化，并提供有价值的内容给你的受众。同时，避免使用违反搜索引擎准则的黑帽SEO技术，以免受到搜索引擎的惩罚。
