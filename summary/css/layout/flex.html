        # 弹性（flex）布局

        教程：http://www.ruanyifeng.com/blog/2015/07/flex-grammar.html

        兼容性越来越好，移动端布局首选，web端布局在对浏览器兼容要求不高的情况下也是首选

        # 容器属性：

        • flex-direction 决定主轴方向，即项目的排列方向
          • row 主轴水平，起点在左端 row-reverse主轴水平，起点在右端
          • column主轴垂直，起点在上方 column-reverse主轴垂直，起点在下方
        • flex-wrap 默认项目都排在一条轴线上，定义如何换行
          • nowrap (默认) 不换行
          • wrap 换行，第一行在上方 wrap-reverse 换行，第一行在下方
        • flex-flow 是上面两个属性的简写形式，默认row wrap
        • justify-content定义项目在主轴上的对齐方式
          • flex-start 主轴起点对齐 flex-end 主轴终点对齐
          • center 主轴居中
          • space-between 两端对齐，项目之间的间隔都相等
          • space-around 每个项目两侧的间隔相等。所以项目之间的间隔比项目与边框的间隔大一倍
        • align-items 定义项目在交叉轴上如何对齐
          • flex-start 交叉轴起点对齐 flex-end 交叉轴终点对齐
          • center 交叉轴居中
          • baseline 项目第一行文字的基线对齐
          • stretch 默认值 如果项目未设置高度或设为auto，将占满整个容器的高度
        • align-content 定义了多根轴线的对齐方式。如果项目只有一根轴线，属性不起作用
          • flex-start 交叉轴起点对齐 flex-end 交叉轴终点对齐
          • center 交叉轴居中
          • space-between 与交叉轴两端对齐，轴线之间的间隔平均分布
          • space-around 每根轴线两侧的间隔相等。轴线之间的间隔比轴线与边框的间隔大一倍
          • stretch 默认值 轴线占满整个交叉轴

        # 项目属性：

        • order 定义项目的排列顺序 数值越小排列越靠前
        • flex-grow 定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大
        • flex-shrink 定义项目的缩小比例，默认为1，即如果空间不足，该项目将所辖
        • flex-basis 定义了在分配多余空间之前，项目占据的主轴空间
        • flex 是上面三个属性的简写形式
        • align-self 允许单个项目与其他项目不一样的对齐方式，可覆盖align-items属性

        # flex实现三栏布局

        <div class="flex-header">HEADER 内容</div>
<div class="flex-container">
  <div class="flex-left">LEFT 内容</div>
  <div class="flex-middle">MIDDLE 内容</div>
  <div class="flex-right">RIGHT 内容</div>
</div>
<div class="flex-footer">FOOTER 内容</div>

        .flex-header,.flex-footer,.flex-container {
  font-size: 30px;
  font-weight: bolder;
}
.flex-header {
  background: cyan;
  height: 100px;
  text-align: center;
}
.flex-footer {
  background: darkgray;
  height: 100px;
  text-align: center;
}
.flex-container {
  display: flex;
}
.flex-left,.flex-right {
  width: 200px;
  height: 300px;
  background: darkturquoise;
}
.flex-middle {
  width: calc(100% - 400px);
  height: 300px;
  background: deeppink;
}
