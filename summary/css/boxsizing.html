          # css盒模型

          # 标准盒模型

          • 从内到外分别是：元素内容(content)、内边距(padding)、边框(border)、外边距(margin)
          • 设置元素的width和height是content部分的宽和高

          # 怪异盒模型

          • IE的怪异盒模型：元素内容(content)部分包含了边框(border)和内边距(padding)
          • 设置元素的width和height是content部分的宽和高

          # box-sizing

          • 定义如何计算一个元素的总宽度和总高度
          • 默认值：content-box 元素的宽高只包含元素内容(content)部分
          • 可选值：border-box 元素的宽高包含元素内容(content)、边框(border)、内边距(padding)
          • 常用值：border-box 简化元素宽高的处理工作，消除了在布局过程中的很多陷阱

          # 示例代码

          • content-box：实际整个元素的宽高是120 + 30 * 2 = 180·
          <div class="content-box">
    <div>div1</div>
    <div>box-sizing: content-box</div>
    <div>padding: 30px; width: 120px; height: 120px</div>
</div>

          .content-box {
  box-sizing: content-box;
  padding: 30px;
  width: 120px;
  height: 120px;
  color: red;
  font-size: 16px;
  text-align: center;
  background: green;
}
          • border-box：实际整个元素的宽高是120 内容超出了元素的高度
          <div class="border-box">
    <div>div2</div>
    <div>box-sizing: border-box</div>
    <div>padding: 30px; width: 120px; height: 120px</div>
</div>

          .border-box {
  box-sizing: border-box;
  padding: 30px;
  width: 120px;
  height: 120px;
  color: red;
  font-size: 16px;
  text-align: center;
  background: blue;
}
