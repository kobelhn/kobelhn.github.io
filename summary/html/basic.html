# HTML

常见元素及属性的使用与详细解释参考：https://www.w3school.com.cn/html/index.asp

WEB开发中，熟练掌握了html知识，可以更准确的使用所需要的元素，更合理的规划和书写html文档树结构。合理的文档树结构，在添加css或者js代码时，可以帮助更合理和快速的实现想要的样式和动态表现等。现代WEB前端框架中的引用或者模版语法等，都有和html极大的类似之处，熟练掌握了html，举一反三可以熟练使用和理解现代WEB前端框架中的引用组件或者模版语法。

# HTML是什么？

HTML is the standard markup language for Web pages.

HTML是网页的标准标记语言。它是超文本标记语言，由一系列能标记内容的元素组成，元素组合描述网页结构，并告知浏览器如何显示内容，是用于创建网页的标准标记语言。

<div></div> <p></p> <img /> <br /> <input /> <span></span> <label></label>

# 基础

<!-- 所有的HTML文档都必须以文档类型声明开头。文档类型声明即告知浏览器如何解析此文档。 -->
<!Doctype html>
<!-- HTML文档以html开头与结尾，文档的可见部分内容在body元素内 -->
<html><body>content</body></html>

# HTML元素

<!--
	HTML元素是从开始标记到结束标记的所有内容
		结束标记和开始标记必须成对出现，部分元素没有结束标记也可以正常显示。
		没有内容的元素为空元素如：<br>；元素可以嵌套。
		HTML元素分为块元素和行内元素，块元素独占一行，行内元素共享一行。
		HTML语义化元素能够清楚的描述其意义给浏览器和开发者。
-->
<tagname>content is here</tagname>

# HTML属性

<!--
	所有的HTML都可以具有属性
		属性提供有关元素的其他信息，始终在开始标记中指定。
		属性通常以名称/值对的形式出现。
-->
<a href="https://www.google.com"></a>
