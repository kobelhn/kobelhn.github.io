# WEB存储

# sessionStorage 和 localStorage 的用法和区别

# 区别

  • window.localStorage - 存储没有截止日期的数据
  • window.sessionStorage - 针对一个 session 来存储数据（当关闭浏览器标签页时数据会丢失）

# 使用方法

  • storage.setItem(key, value) 设置
  • storage.getItem(key) 获取
  • storage.removeItem(key) 删除
