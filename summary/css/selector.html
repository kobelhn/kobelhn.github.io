                    # :root

                    :root这个伪类匹配文档树的根元素。对于HTML来说，:root表示<html>,除了优先级更高之外，与html选择器相同。

                    在生命CSS变量时，:root很有用

                    :root {
  --main-color: hotpink;
  --pane-padding: 5px 42px;
}
