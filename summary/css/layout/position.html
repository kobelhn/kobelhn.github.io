      # 定位（position）布局

      # position

      • static 可选值 元素使用正常的布局行为
      • relative 可选值 元素相对未添加定位时在文档中的位置定位
      • absolute 可选值 元素会被移出文档流，相对于最近的非static定位的祖先元素位置定位 找不到则相对于body定位
      • fixed 可选值 元素会被移出正常文档流，相对于屏幕视口（viewport)的位置定位
      • sticky 可选值 元素根据正常文档流进行定位，然后相对它的最近滚动祖先和最近块级祖先定位
      <div class="position-relative">
  <div class="position-absolute">absolute定位的元素</div>
</div>
<div class="position-absolute">absolute定位的元素</div>
<div class="position-fixed">fixed定位的元素</div>

      .position-relative {
  position: relative;
}
.position-absolute {
  position: absolute;
  width: 100px;
  height: 60px;
  top: 0;
  right: 0;
  background: cyan;
}
.position-fixed {
  position: fixed;
  width: 100px;
  height: 60px;
  bottom: 0;
  right: 0;
  background: cyan;
}

      # position实现三栏布局

      <div class="position-header">HEADER 内容</div>
<div class="position-container">
  <div class="position-middle">MIDDLE 内容</div>
  <div class="position-left">LEFT 内容</div>
  <div class="position-right">RIGHT 内容</div>
</div>
<div class="position-footer">FOOTER 内容</div>

      .position-header,
.position-footer {
  height: 100px;
  font-size: 30px;
  font-weight: bolder;
  background: green;
  text-align: center;
}
.position-footer {
  background: cyan;
}
.position-container {
  position: relative;
}
.position-left,
.position-right {
  position: absolute;
  width: 200px;
  height: 300px;
  top: 0;
  font-size: 30px;
  font-weight: bolder;
}
.position-middle {
  height: 300px;
  margin: 0 200px;
  background: steelblue;
  font-size: 30px;
  font-weight: bolder;
}
.position-left {
  left: 0;
  background: yellowgreen;
}
.position-right {
  right: 0;
  background: burlywood;
}
